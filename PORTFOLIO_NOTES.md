# 作品集说明

## Showcase 文案

移动端 MBTI 测试原型，包含逐题答题、状态管理、16 型结果解析与维度图表可视化，展示从需求到测试的完整前端实现。



1. 我用 Codex 先做需求澄清，把 MBTI 测试拆成页面流程、题目结构、答题记录、计分规则和结果映射，确保项目范围适合作品集展示。
2. 我用 Codex 辅助实现集中式状态管理，用一个 `state` 对象驱动当前题号、答题记录、完成状态、重测和结果跳转，降低交互逻辑出错概率。
3. 我用 Codex 生成测试方案和纯 JavaScript 自动化测试脚本，通过 ENFJ、ISTP、INFP 三组模拟路径验证计分函数，提升结果页展示可信度。
